# Data Sources

## Primary Source

1. Statistics Canada Table 13-10-0701-01
   - Source page: https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1310070101
   - Dataset title: Waiting times at selected percentiles for specialized services, by type of service, household population aged 15 and over, Canada
   - Geography: Canada
   - Release date on source page: 2015-12-17
   - Frequency: Occasional

## Why It Works

- It is a Canadian public dataset from a primary source.
- It supports a healthcare access story instead of a generic dashboard.
- It lets us analyze service-level differences in wait-time distributions.

## Analysis Notes

- Use service type as the main comparison dimension.
- Focus on percentile comparisons to show long-tail delays and service variability.
- Pair charts with business-facing recommendations about access pressure and prioritization.
