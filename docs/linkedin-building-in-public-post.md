# LinkedIn Post: Building In Public

Here is a polished LinkedIn post you can use to talk about the portfolio as ongoing work.

## Draft

I’m currently building a healthcare-focused Business Analyst portfolio using Canadian public data.

Instead of taking another random course, I wanted to create proof of work that actually looks relevant to the Canadian job market.

So far I’ve built:
- a healthcare wait-times analysis using Statistics Canada data
- a healthcare access coverage project using facility and population data
- SQL and SQLite workflows that generate dashboard-ready outputs
- a local frontend tracker to monitor the full project journey

A few of the insights from the current output:
- non-emergency surgeries showed the strongest long-tail delay, with a 95th percentile wait of 34.7 weeks
- Prince Edward Island had the highest observed facility coverage in the current province-level output at 52.01 facilities per 100,000 residents
- Alberta was lowest in the current comparison output at 5.91

I’m now turning these outputs into Power BI dashboards, resume material, and application-ready stories.

This has been a good reminder that for entry-level analyst roles, the goal is not to collect more certificates.
The goal is to show that you can work on real business problems, with real data, and communicate something useful.

I’ll keep sharing the progress as I build.
