## Projects

### Canadian Healthcare Wait Times Analysis
- Built a healthcare wait-times analysis workflow using Statistics Canada data and SQLite to compare median and tail delays across specialist visits, diagnostic tests, and non-emergency surgeries.
- Identified non-emergency surgeries as the longest-delay service group in the current output, with a 95th percentile wait time of 34.7 weeks.
- Prepared KPI-ready outputs for a Power BI-style dashboard focused on percentile-based service comparison.

### Canadian Healthcare Access Dashboard
- Combined Canadian healthcare facility and population data to create a province-level access coverage dataset for dashboard reporting.
- Generated normalized facility-per-100k metrics to compare healthcare access coverage across provinces.
- Highlighted strong regional differences in current output, including Prince Edward Island at 52.01 facilities per 100,000 residents and Alberta at 5.91.
