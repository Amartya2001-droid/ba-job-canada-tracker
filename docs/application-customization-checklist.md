# Application Customization Checklist

## Before You Apply
- Read the full job description once for responsibilities
- Read it again for tools and keywords
- Confirm at least 70% stack overlap
- Decide which project is the stronger fit

## Resume Customization
- Match the headline to the role language
- Move the most relevant project bullet higher
- Use wording from the posting where it truthfully matches your work

## Cover Note Customization
- Mention the role title exactly
- Mention one reason the company or team fits your interest
- Reference one project that matches the job problem

## Final Submission Check
- Resume file name is clean
- Contact details are correct
- No generic company name remains in the note
- Application is logged in `tracker/applications.csv`
