# LinkedIn Post Draft: Project 2

I built a Canadian healthcare access analysis using healthcare facility data and population estimates from public Canadian sources.

Question I wanted to answer:
Which provinces appear more or less covered when healthcare facilities are normalized by population size?

What I found:
- Prince Edward Island had the highest observed facility coverage in the current output at 52.01 facilities per 100,000 residents.
- Saskatchewan ranked second at 31.02.
- Alberta was lowest in the current comparison output at 5.91.

Tools used:
- SQLite
- SQL
- Dashboard planning for Power BI

This project helped me practice combining supply-side and population context to build a more decision-ready view of regional healthcare access.
